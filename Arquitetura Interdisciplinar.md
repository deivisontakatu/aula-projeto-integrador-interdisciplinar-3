# 🧠 Arquitetura Interdisciplinar do PI3

## 🎯 Conceito Estruturante do Projeto

**Sistema Mobile Integrado à IoT Industrial com Arquitetura em Nuvem,
Interfaces Industriais de Alta Performance e Gestão Ágil via Scrum**

O projeto terá como núcleo:

📡 Integração IoT + Backend → ☁️ Nuvem → 📱 Mobile → 🖥 Interfaces
Industriais\
🔁 Gestão via Scrum + Integração Vertical e Horizontal
